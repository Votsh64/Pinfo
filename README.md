# Pinfo

Lightweight web-based monitoring interface for your Raspberry Pi

Page de présentation : [pinfo.mgeek.fr](http://pinfo.mgeek.fr)

- - -

##Installation en moins de deux minutes

L'installation de Pinfo se fait en seulement 3 commandes ! 

Tout d'abord, il vous faut un environnement web/php.
Le plus simple d'utilisation est Apache, il s'installe ainsi :

    sudo apt-get install apache2 php5 libapache2-mod-php5

 
Vous n'avez plus qu'a télécharger Pinfo et à l'extraire dans votre racine web (/var/www par défaut)

    cd /var/www
    wget https://github.com/mGeek/Pinfo/archive/master.zip
    unzip master.zip
